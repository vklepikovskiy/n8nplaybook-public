## üõ† How to Use These Workflows

All workflows are located in the `/workflows` directory. You can use them in two ways:

### Option 1: Direct Import (Recommended)
1. Navigate to the `workflows` folder in this repo.
2. Open the desired `.json` file.
3. Click the **"Raw"** button.
4. Copy the URL from your browser's address bar.
5. In your **n8n** instance, go to **Import from URL** and paste the link.

### Option 2: Copy-Paste
1. Open the `.json` file here on GitHub.
2. Copy the file content.
3. In your **n8n** canvas, simply press `Ctrl+V` (or `Cmd+V`) to paste the nodes.
